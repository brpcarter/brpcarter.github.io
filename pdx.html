PDX

2017 - 2018

Developed roof concepts for the $1.3bn expansion of Portland International Airport (PDX). Provided inspiration from form-finding and structural analysis to the rest of the project team. These images come from an analytical model constructed based on BRG’s Thrust Network Analysis (Philippe Block, ETHZ) to construct funicular geometry. In this case a continuous catenary roof system has been discretized into units resembling trees. Project ongoing.
